# World context

Background and source material on the deck's topic. This is **input** to the
spec — the facts and raw material the deck's argument draws on. It is *not* the
spec itself (that's `deck/`).

## How to use

- Read the relevant file **before** writing slide copy or asserting a fact.
- When a slide makes a claim sourced from here, cite the file (traceability).
- Curated by Isaiah. Don't edit these unless asked — add new material as new
  files, and add a line to the index below.
